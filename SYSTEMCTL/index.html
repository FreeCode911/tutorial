Fixing Systemctl

The default `systemctl` does not work with this system. Follow these steps to fix it:

Step 1: Install Python 3

apt-get install python3

Ensure you have `python3` installed on your system. This is required for the `systemctl` replacement script.

Step 2: Download Systemctl Replacement Script

curl -o /bin/systemctl https://raw.githubusercontent.com/gdraheim/docker-systemctl-replacement/master/files/docker/systemctl3.py

Use the `curl` command to download the replacement `systemctl` script from the provided URL.

Step 3: Make the Script Executable

chmod +x /bin/systemctl

Grant execute permissions to the `systemctl` script so that it can be run on your system.

Step 4: Verify the Installation

systemctl

Run the `systemctl` command to ensure that the replacement script is functioning correctly. You should see a list of services if everything is set up properly.
